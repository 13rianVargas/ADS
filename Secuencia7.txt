@startuml
actor Usuario
participant Kiosco
participant RecursoControlador
participant RecursoDAO
participant PrestamoDAO
participant ConexionBD

Usuario -> Kiosco : Seleccionar opción "Devolver recurso"
Kiosco -> Usuario : Solicitar colocar recurso en escáner
Usuario -> Kiosco : Coloca recurso
Kiosco -> RecursoControlador : Escanear recurso y obtener ID
RecursoControlador -> RecursoDAO : Validar que el recurso pertenezca a la biblioteca
RecursoDAO -> ConexionBD : Consulta de recurso por ID
ConexionBD --> RecursoDAO : Datos del recurso
RecursoDAO --> RecursoControlador : Recurso válido
RecursoControlador -> Kiosco : Mostrar información del recurso

Usuario -> Kiosco : Confirmar devolución
Kiosco -> RecursoControlador : Confirmación recibida
RecursoControlador -> PrestamoDAO : Registrar devolución (fecha y hora)
PrestamoDAO -> ConexionBD : Actualizar estado del recurso
ConexionBD --> PrestamoDAO : Confirmación de actualización
PrestamoDAO --> RecursoControlador : Registro exitoso
RecursoControlador -> Kiosco : Mostrar mensaje de devolución exitosa
@enduml
